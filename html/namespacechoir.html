Detailed Description

The choir namespace contains the code specific to the ZGChoir demonstration application.

The choir namespace is a superset of the zg namespace, which is itself a superset of the muscle namespace

anonymous enum

Our enumeration of supported notes, and their numeric indices.

These can be used to compute bit-positions within our 64-bit chord values. Note that we only define 21 notes for now; the rest of the 64 bits in each chord are reserved for future expansion.
